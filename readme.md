# 🎵 Musicify - Spotify UI Clone

A modern Spotify-inspired music streaming interface built using **React, Vite, and Tailwind CSS**.

Musicify recreates the look and feel of a modern music streaming platform through responsive layouts, reusable UI components, and a clean dark-themed design system.

## 🌐 Live Demo

https://ayeshaas-sc.netlify.app/

---

## ✨ Features

- Spotify-inspired streaming interface
- Responsive dark-themed layout
- Trending songs section
- Popular artists showcase
- Album and playlist sections
- Sidebar navigation system
- Reusable React components
- Interactive hover-based UI experience

---

## 🚀 Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Icons
